# 大题整理



## 1. 如何认识习近平新时代中国特色社会主义思想的重大意义？

1. 习近平新时代中国特色社会主义思想，以一系类原创性战略性的重大思想丰富和发展了马克思主义，是当代中国马克思主义、21世纪马克思主义
2. 习近平新时代中国特色社会主义思想，是对马克思列宁主义、毛泽东思想、邓小平理论、“三个代表”重要思想、科学发展观的继承和发展、是马克思主义中国化的最新成果，是中国特色社会主义理论体系的重要组成部分
3. 习近平新时代中国特色社会主义思想，是当和人民实现经验和集体智慧的洁净
4. 习近平新时代中国特色社会主义思想，是全党全国人民为实现中华民族伟大复兴而奋斗的行动指南

## 2. 从全面深化经济体制改革的角度，如何认识完善社会主义市场经济体制的重点和难点？

1992年，党的十四大提出了我国经济体制改革的目标是建立社会主义市场经济体制。

完善社会主义市场经济体制的重点和难点是：处理好政府和市场的关系，要使市场在资源配置中其决定性作用，更好发挥作用。（书上）

从激发市场主体活力角度，如何处理好政府与市场的关系？

转变政府职能，积极简政放权，更好发挥政府作用

完善产权保护制度，深化产权制度改革

全面实现市场准入负面清单制度，清理废除妨碍统一市场和公平竞争的各种规定和做法

完善主义由市场决定价格的机制，加快要素价格市场化改革，使市场在资源配置中其决定性作用。价格机制是市场机制的核心，市场决定价格是市场在资源配置中起决定性作用的关键

重点和难点：计划与市场之间的关系问题，政府与企业之间的关系问题，国有经济与民营经济之间的关系问题 （网上）

## 3. 结合我国民主政治的发展历程，谈谈如何利用制度体系保障人民当家作主？

进入近代社会以来，中国人民尝试了包括君主立宪制、议会制、多党制、总统制在内的各种政治制度，但是没有找到一条合适中国国情的政治道路，反而使中国陷入军阀混战、四分五裂、外敌入侵的状态。在总结历史经验后，要坚持走中国特色社会主义政治发展道路。

要健全人民当家作主的制度体系：发展和完善人民代表大会制度；推动协商民主广泛、多层、制度化发展；积极稳妥地推进政治体制改革

## 4. 依据马克思有关人与自然关系的理论，怎样建设人与自然和谐共生的现代化？

马克思观点：人依赖于自然，靠自然生活；人可以认识自然，改造自然，使自然界为自己服务；最后人的活动必须遵循自然规律，绝不能凌驾于自然之上。

怎么建设：要走出一条区别于传统现代化的新道路，将绿色理念贯彻落实于现代化的各领域、全过程，绝不走“先污染后治理的老路”。一是以绿色理念引领新型工业化；二是让绿色化贯穿农业现代化发展始终；三是让城市融入大自然。

## 5. “一带一路”战略对推动形成全面开放格局和构建人类命运共同体的重大意义？

一带一路意义（216页）：

一带一路成为破解人类发展难题的新平台；一带一路成为深化中国同周边国家经贸、人文等领域交流合作的新平台；一带一路成为探索和创新全球经济治理新模式的重要平台。以一带一路建设为代表的国际合作新平台的打造，改写了全球经济版图和政治格局，为世界添加了共同发展的动力，引导了全球化和国际化关系构建的新方向。

构建人类命运共同体的意义（好像没啥联系啊）：

第一，构建人类命运共同体思想是对新中国不同时期重大外交思想和主张的继承和发展；第二，构建人类命运共同体思想反映了中外优秀文化和全人类共同价值追求；第三，构建人类命运共同体思想适应了新时代中国与世界关系的历史性变化；第四，构建人类命运共同体思想指明看世界发展和人类未来的前进方向。

一带一路意义（94页）：

首先，一带一路建设是我国扩大对外开放的重大举措和经济外交的顶层设计；其次，一带一路建设是为破解人类发展难题提供了中国指挥和中国方案；再次，一带一路建设是探索全球经济治理新模式、构建人类命运共同体新平台；最后，一带一路建设是新时代中国特色社会主义的伟大开放实践。
